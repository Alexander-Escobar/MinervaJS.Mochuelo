MinervaJS.Mochuelo

MinervaJS.Mochuelo es un módulo para Node.js diseñado para la gestión del Modelo de Datos, especialmente útil para desarrollos basados en la filosofía "Base de Datos Primero".

Funcionalidades Principales

Este módulo te permite gestionar tu modelo de datos a través de diversas funciones:

  • loadModels - Lista las Entidades (Tablas) y carga los modelos desde el archivo JSON. Por defecto, busca el archivo models.json.
  • getModel - Obtiene un modelo específico por su nombre, proporcionando detalles como sus atributos (columnas) y otras características (clave primaria, clave foránea, columnas, longitud y precisión).
  • addModel - Agrega un nuevo modelo tanto al archivo JSON persistente como a la instancia de ModelManager en memoria. Permite adicionar metadatos o características personalizadas dentro de las entidades o junto a sus atributos.
  • updateModel - Actualiza un modelo existente, tanto en el archivo JSON como en la instancia de ModelManager.
  • deleteModel - Elimina un modelo especificado del archivo JSON y de la instancia de ModelManager.
  • addColumnAttribute - Agrega un nuevo atributo a una columna específica dentro de la definición de un modelo.
  • editModelAttribute - Edita un atributo existente de una columna específica dentro de la definición de un modelo.
  • deleteColumnAttribute - Elimina un atributo de una columna específica dentro de la definición de un modelo.
  • hasColumnAttribute - Verifica si una columna específica dentro de la definición de un modelo posee un atributo determinado.
  • Crear Objetos o Clases basados en las entidades del Modelo - IMPLEMENTADO. Permite la manipulación de tuplas (Registros) basada en la estructura del modelo.
    Ejemplo de uso disponible (ver documentación del módulo).

Toda la información y la estructura del modelo de datos se guardan y gestionan en un archivo con formato .JSON.

Instalación

Para comenzar a utilizar MinervaJS.Mochuelo en tu proyecto de Node.js, puedes instalarlo fácilmente usando npm:

$ npm i minervajs-mochuelo
